Gizmoball Details

Project Information:

This project was a group project completed in my fourth year. It set out to use java to create a pinball simulator. The MVC design pattern was used throughout the project. Java Swing was used in order to create an interative GUI for the project, this involved both a 'build' interface for the user to build their own pinball design and a 'run' interface to run the simlation. The project was tested using Junit testing as well as user testing.
