Project information

The objective of my final year project was to create a secure online interactive training resource which will assist health practitioners to assess childrenâ€™s motor skills. Additionally it was designed to provide an online interactive training resource.

A key requirement was to build an application with future growth in mind. To enable this a dynamic database model was designed to allow system maintenance to be configurable and hence a non-technical task. Industry strength tools were used to develop the application and a common set of software design patterns were adopted within the build. This included using the CodeIgniter4 framework which follows a MVC design principle. After researching the best framework to use, to rapidly build a substantial number of web pages/forms I self taught myself to use this framework. Quality assurance of the software was ensured by using a variety of testing techniques including automated unit testing. Feedback from professional health practitioners was used to help inform the final product.
